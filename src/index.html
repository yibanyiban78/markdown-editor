极简Markdown编辑器

轻量 · 快速 · 所见即所得 · 开源免费

🚀 快速开始

  • 点击工具栏 📂 打开一个 .md 文件
  • 点击 📄 新建空白文档
  • 直接将 .md 文件拖拽到窗口中
  • 使用 Ctrl+O 快速打开文件

⌨️ 快捷键

Ctrl + N新建文件
Ctrl + O打开文件
Ctrl + S保存文件
Ctrl + F搜索
Ctrl + =放大字体
Ctrl + -缩小字体
Ctrl + Shift + O切换大纲面板
Escape关闭搜索栏

✨ 功能特色

  • 🔤 语法高亮 — 支持 190+ 编程语言
  • 📐 数学公式 — KaTeX 行内/独立公式
  • 📊 图表绘制 — Mermaid 流程图/时序图/甘特图
  • 🌙 深色/浅色主题 — 一键切换，自动记忆
  • 📑 大纲面板 — 自动提取目录，点击跳转
  • 📤 导出 HTML/PDF — 方便分享
