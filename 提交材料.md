## 二、软件介绍

### 一句话简介
一款轻量、快速、所见即所得的桌面 Markdown 编辑器，支持实时预览、代码高亮、数学公式和图表绘制。

### 详细描述（300-500字）

极简Markdown编辑器是一款基于 Electron 构建的桌面端 Markdown 编辑工具，致力于提供流畅、高效的写作体验。

**主要功能：**

- **三种编辑模式**：纯预览模式、双栏同步滚动模式（左预览右源码）、纯源码模式
- **实时 Markdown 渲染**：基于 marked 引擎，支持 GFM 标准语法
- **代码语法高亮**：基于 highlight.js，覆盖 190+ 编程语言
- **数学公式支持**：集成 KaTeX，支持行内公式 `$...$` 和独立公式 `$$...$$`
- **图表绘制**：集成 Mermaid，支持流程图、时序图、甘特图等
- **文件管理**：支持打开/新建/保存 .md 文件，支持拖拽打开
- **自动保存**：编辑后 1.5 秒自动存盘（可关闭）
- **全文搜索**：支持匹配高亮和上下跳转
- **文档导出**：支持导出为 HTML 和 PDF 格式
- **深色/浅色主题**：一键切换，自动记忆偏好
- **大纲面板**：自动提取 H1-H3 标题，点击可跳转
- **字体调节**：编辑器字号 11px–24px 自由调节
- **字数统计**：实时显示中文字数和总字符数

**软件特色：**

- 纯本地处理，不上传任何用户数据，隐私安全有保障
- 开源免费，代码完全透明
- 轻量高效，启动迅速，运行流畅

### 适用场景

- 程序员编写技术文档、README、博客文章
- 学生记笔记、写论文
- 日常写作、待办清单管理
- 技术团队内部知识管理

## 五、提交流程（通用步骤）

1. **注册账号** — 在相应开放平台注册开发者账号（个人即可）
2. **创建应用** — 填写软件名称、分类、简介
3. **上传安装包** — 上传 setup.exe 安装包
4. **填写信息** — 复制粘贴上方的软件介绍文案
5. **上传截图** — 上传 assets/ 目录下的截图
6. **提交审核** — 提交后等待审核（通常 3-10 个工作日）
