# 隐私政策

**更新日期：2026 年 6 月 14 日**

欢迎使用"极简Markdown编辑器"（以下简称"本软件"）。本软件是一款基于 Electron 框架开发的桌面端 Markdown 编辑工具。我们深知隐私对您的重要性，并承诺严格遵守相关法律法规，保护您的个人信息安全。

## 1. 信息收集与使用

### 1.1 本软件不收集的信息
本软件**不会**收集以下任何信息：
- 个人身份信息（姓名、身份证号、手机号、邮箱等）
- 位置信息
- 通讯录、相册等个人数据
- 设备识别码、MAC 地址等硬件信息
- 网络浏览记录
- 任何形式的生物识别信息

### 1.2 本软件处理的信息
本软件仅在本地处理您的数据，具体包括：

- **您编辑的 Markdown 文件内容**：所有文件内容仅在您的本地设备上处理，不会被上传至任何服务器
- **应用设置偏好**：包括主题选择（深色/浅色）、字体大小、自动保存开关等偏好设置，仅存储在您设备的本地 localStorage 中
- **自动保存的草稿**：启用自动保存功能后，编辑内容暂存于本地存储，不会被传输到外部

## 2. 第三方服务

本软件使用了以下 CDN 托管的开源库，仅在应用启动时从 CDN 加载：

| 库名称 | 用途 | CDN 地址 |
|--------|------|---------|
| marked | Markdown 解析渲染 | cdn.jsdelivr.net |
| highlight.js | 代码语法高亮 | cdnjs.cloudflare.com |
| KaTeX | 数学公式渲染 | cdn.jsdelivr.net |
| Mermaid | 图表渲染 | cdn.jsdelivr.net |

上述 CDN 服务商可能有各自的日志记录策略（如请求 IP、时间等），这不在本软件的控制范围内。您可以选择断开网络后使用本软件（核心功能无需联网）。

## 3. 数据存储与安全

- 所有文件内容存储在您自己选择的本地路径中
- 应用设置以键值对形式存储在 Electron 的本地存储中
- 本软件不会将任何数据传输至外部服务器

## 4. 隐私政策的更新

我们可能会不时更新本隐私政策。更新后的隐私政策将通过 GitHub 仓库发布。重大变更时，我们会在软件内进行提示。
