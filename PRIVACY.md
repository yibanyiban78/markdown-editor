# 隐私政策

**更新日期：2026 年 6 月 15 日**

欢迎使用"极简Markdown编辑器"（以下简称"本软件"）。本软件是一款基于 Electron 框架开发的桌面端 Markdown 编辑工具。我们深知隐私对您的重要性，并承诺严格遵守相关法律法规，保护您的个人信息安全。

## 1. 信息收集与使用

### 1.1 本软件不收集的信息
本软件**不会**收集以下任何信息：
- 个人身份信息（姓名、身份证号、手机号、邮箱等）
- 位置信息
- 通讯录、相册等个人数据
- 设备识别码、MAC 地址等硬件信息
- 网络浏览记录
- 任何形式的生物识别信息

### 1.2 本软件处理的信息
本软件仅在本地处理您的数据，具体包括：

- **您编辑的 Markdown 文件内容**：所有文件内容仅在您的本地设备上处理，不会被上传至任何服务器
- **应用设置偏好**：包括主题选择（深色/浅色）、字体大小、自动保存开关等偏好设置，仅存储在您设备的本地 localStorage 中
- **自动保存**：对已经由您打开或保存过的文件，启用自动保存后会直接写回该本地文件；新建或拖入的文档仍需由您首次确认保存位置

## 2. 第三方服务

本软件随安装包内置以下开源库，应用启动和 Markdown 渲染不会从 CDN 下载代码：

| 库名称 | 用途 |
|--------|------|
| marked | Markdown 解析渲染 |
| DOMPurify | HTML 内容安全净化 |
| highlight.js | 代码语法高亮 |
| KaTeX | 数学公式渲染 |
| Mermaid | 图表渲染 |

Markdown 文档如果包含远程图片（例如 `https://` 图片地址），预览和导出时可能向该图片服务器发起请求。点击文档中的外部链接时，链接会交给系统默认浏览器打开。除这些由文档内容触发的网络访问外，本软件的编辑、预览和保存功能可离线使用。

## 3. 数据存储与安全

- 所有文件内容存储在您自己选择的本地路径中
- 应用设置以键值对形式存储在 Electron 的本地存储中
- 本软件不会将任何数据传输至外部服务器

## 4. 隐私政策的更新

我们可能会不时更新本隐私政策。更新后的隐私政策将通过 GitHub 仓库发布。重大变更时，我们会在软件内进行提示。
